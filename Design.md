## **1) 목표 & 제품 방향**

- 1.1) **제품 목표(MVP)**: 프로젝트 단위 **Item을 축적·정리·검색**하게 만든다. 공유/이력서/AI는 차기.
- 1.2) **타깃**: 스스로 성과/이력을 꾸준히 관리하지 못한 사용자.
- 1.3) **제품 철학**: *Guided by default, Expert on demand*
    - 기본은 **의견 있는(opinionated) 가이드 모드**로 행동 유도
    - 필요 시 **자유 편집 모드**로 전환 가능
- 1.4) **핵심 축**: Item CRUD/필터 1순위. 성과·스킬·경력은 **Item과 연결**하여 맥락 강화.

## **2) 범위(스코프)**

- 2.1) **In (MVP)**: Item 만들기/편집, 목록·검색·필터, 홈 대시보드(기간 목록·미완 큐·타임라인·텍스트 TODO), 저장된 보기(동적 묶음), Backfill, 복제(Fork)
- 2.2) **Out (차기)**: 외부 공유, 이력서 포맷/PDF, 협업(코멘트/권한), AI 요약/추천

## **3) 정보 구조(IA)**

- 3.1) **홈(대시보드)**: 기간별 Item 목록, 미완/보완 큐, 텍스트 TODO, 타임라인(겹침 시각화·공백 강조)
- 3.2) **라이브러리**: Item 목록/검색/필터, 저장된 보기(동적 묶음)
- 3.3) **Item 상세**: 필수·선택 필드, 연결(성과/스킬/경력), 증거 링크
- 3.4) **설정**: 분류/태그 관리, 힌트/가이드 노출 강도(가이드 모드 토글)

## **4) 홈 대시보드**

- 4.1) **첫 노출**: **기간별 Item 목록**(월/분기 토글) + 뷰 전환(카드/리스트)
    - **AC**: 홈 진입 시 기간 목록 최상단 고정, 개인별 정렬/전환 상태 기억
- 4.2) **미완/보완 큐**: 초안·필수 누락·증거 링크 없음 정렬
    - **AC**: 큐에서 1클릭 보완 진입
- 4.3) **텍스트 TODO**: 간단 메모, (선택) Item 링크/기한/핀
    - **AC**: Enter로 추가, 드래그 정렬, 핀은 상단 고정
- 4.4) **타임라인**: 12~24개월, **겹침 허용·표현만**, 공백 강조
    - **AC**: 월/분기 토글, “겹치는 기간만 보기” 필터, 타임라인↔목록 스크롤 싱크

## **5) 핵심 사용자 흐름**

- 5.1) **Backfill(공백 채우기)**
    - 공백 클릭 → 미니 폼(제목/기간/소속) → **초안 저장** → (기본) 보완 패널 안내
    - **AC**: 공백 클릭→저장 ≤ 2클릭, 저장 즉시 홈 위젯 동기 반영
- 5.2) **반복 업무 기록(복제/Fork)**
    - 기존 Item **복제** → 기간/맥락 갱신 → 저장(제목 “(복제)” 임시)
    - **AC**: 복제→저장 2클릭, 필수 미충족 시 미완 큐 표기
- 5.3) **정리(동적 묶음/저장된 보기)**
    - 필터 조합 → **보기 저장** → 홈 단축
    - **AC**: 필터→저장 1회 클릭, 카드에 아이템 수/마지막 갱신 표기
- 5.4) **검색/보기**
    - 통합 검색(제목/요약/태그/소속/키워드) + 필터(기간/태그/상태/지표 보유/겹침 보기)
    - **AC**: 입력 후 300ms 내 프리뷰, 칩 토글 즉시 반영, 정렬(최근 수정/기간/임팩트)

## **6) 표현**

- 6.1) **표현**: 타임라인 **레이어**로 겹침 시각화, 저장된 보기로 **속성 기반 묶음**
- 6.2) **복제(Fork)**: 반복 업무 기록을 빠르게 쌓기 위한 2클릭 흐름

## **7) Item(프로젝트) 핵심 필드**

- 7.1) **필수**: **제목, 기간, 소속, 기술스택 태그, 핵심 기여 요약(2줄)** (+자유 텍스트 ‘내용’)
    - **AC**: 미충족 시 상태=‘초안’ → 미완 큐 노출
- 7.2) **선택**(STAR 비강제, 일반 명칭): 배경/문맥, 목표/문제, 역할/책임, 주요 작업, 결과(정량), 영향(정성), 증거 링크, 협업, 규모/복잡도, 제약/리스크, 회고, 키워드
- 7.3) **작성 UX**: *짧→길*, 필드별 힌트, 핵심 요약 2줄 가드레일, 중복 허용+검색 강화

## **8) 가이드 모드(행동 유도 강화)**

- 8.1) **완성 체크리스트**: 필수 5개 + **권고 3개(정량 지표 1·증거 링크 1·2줄 요약 충족)**
- 8.2) **다음 베스트 액션(NBA)**: “이 Item을 완료하려면 → 결과 지표 1개부터”
- 8.3) **10분 채우기 모드**: 타이머 기반 미션형 입력(마찰↓)
- 8.4) **템플릿/블루프린트**: 유형별 질문·샘플(성능 개선/신규 기능/장애 대응 등)
- 8.5) **품질 코치 패널**: 실시간 코멘트(정량 권고/링크 제안), **소프트 락**(완성도 라벨)
- 8.6) **예시 라이브러리**: 좋은/나쁜 예시 카드, 문장 틀 가져오기
- 8.7) **습관 유도**: 주간 보완 추천, 라이트 배지(Backfill 스트릭·정량 10개 등)
- 8.8) **완료 정의(DoD)**:
    - 기본 완료 = 필수 5 충족
    - **추천 완료 = 기본 완료 + 정량 1 + 증거 링크 1**(배지/정렬 가중치)

## **9) 대시보드 위젯(우선순위)**

- 9.1) **기간 공백 지도** → 9.2) **미완 큐** → 9.3) **최근 편집/즐겨찾기** → 9.4) **(후순위) 품질 점수 v1**

---

## **10) 성공 지표**

- 10.1) **기간 공백 해소율**(30일) = 공백 클릭 후 24h 내 생성된 Item / 공백 클릭 수
- 10.2) **Backfill 생성 비중** = Backfill로 만든 Item / 전체 신규
- 10.3) **묶음 활용도** = 저장된 보기 신규/재방문(7/30일), 보기 내 클릭률
- 10.4) **검색 효율** = 목표 Item 상세 도달 시간/클릭 수
- 10.5) **보완 전환율** = 미완 큐 진입 → 보완 완료

---

## **11) 실험(가이드 강도 튜닝)**

- 11.1) 힌트 노출 강도(기본 vs 클릭형)
- 11.2) 정량 권고 강도(라벨만 vs 점수 반영)
- 11.3) Backfill 후속(보완 모달 ON vs 토스트)
- 11.4) 템플릿 밀도(1줄 vs 3블록)

---

## **12) 리스크 & 가드레일**

- 12.1) **허수 입력**: 수치 근거 라벨(대시보드/로그/추정)로 투명성
- 12.2) **프라이버시**: 증거 링크 기본 비공개, 공유 단계에서만 공개
- 12.3) **오버-가이드**: Expert 토글, 힌트/체크리스트 숨김
- 12.4) **대시보드 과포화**: 초기엔 공백 지도/미완 큐/최근 편집 중심

## **13) 릴리스 플랜(제안)**

- 13.1) **R0 — Item 만들기·편집**: 필수 5, Backfill 미니 폼, 복제, 미완 큐
- 13.2) **R1 — 목록·검색**: 기간 그룹/정렬/필터, 저장된 보기(동적 묶음)
- 13.3) **R2 — 대시보드**: 공백 지도, 최근 편집, 텍스트 TODO, 타임라인
- 13.4) **R3 — 가이드 강화**: 품질 코치, 템플릿, 추천 완료 배지, 품질 점수 v1
